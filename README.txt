 Trocador de papel de parede, viva a liberdade de expressão visual!

####################################################################

Se o script de logon da rede já tiver finalizado pode fechar essa
janela, caso contrário ela se fechará automaticamente em 6 minutos.

Repositório: https://github.com/evandrojr/ChangeWallPaper

É necessário passar uma imagem via linha de comando ou colocá-la no mesmo
diretório do executável. Em seguida crie um atalho para a aplicação e 
coloque na pasta 'Inicializar' do windows que fica no Menu iniciar.
